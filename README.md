# Moxfield Checkmarker
Adds a simple checklist to Moxfield decks


## Usage
Click "Enable Checklist" to show, "Disable Checklist" to hide.  
Checkmarks appear next to cards, clicking "Checks" them.
Click "Clear" to clear all checked items.
